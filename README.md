# dot-files
Collection of my dot-files

## Requirements

```
sudo apt-get update
sudo apt-get install -y zsh vim
```

## Roll out

```bash
./deploy.sh
```

## Deploy to remote hosts

```
echo "<[user]@host>" >> remote-hosts
```

## Add external repos

```bash
git submodule add <git-path>
```
